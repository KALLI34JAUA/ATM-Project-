ATM Management System

A simple ATM Management System built using Java/PHP/Python (adjust based on your implementation) that simulates the basic functionalities of an Automated Teller Machine. This system allows users to securely manage bank accounts, perform transactions, and view account details in a user-friendly interface.

Features

User Authentication: Secure login with account credentials.

Balance Inquiry: Check current account balance.

Cash Withdrawal: Withdraw funds from the account securely.

Cash Deposit: Deposit funds into the account.

Transaction History: View past transactions and account activities.

PIN Change: Update account PIN/password for security.

Technologies Used

Programming Language: Java / PHP / Python

Database: MySQL / SQLite / File-based (depending on implementation)

Frontend: Console-based / GUI (if applicable)

Development Environment: Eclipse / NetBeans / XAMPP (depending on language)

Installation

Clone or download the repository.

If database is used, import sql/database.sql into your DBMS.

Update the database credentials in the configuration file (if applicable).

Run the main program file:

Java: Main.java

Python: atm.py

PHP: Launch via XAMPP htdocs folder.

Usage

Login using valid account credentials.

Navigate through menu options: Withdraw, Deposit, Balance Inquiry, PIN Change, or Transaction History.

Perform transactions securely and view updated account details.

Logout after completing operations.
